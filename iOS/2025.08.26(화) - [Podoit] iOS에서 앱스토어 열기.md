## 문제 상황

앱 내에서 리뷰 버튼을 눌렀을 때 App Store 페이지로 이동하도록 구현했는데, 시뮬레이터에서 App Store로 열리지 않는 문제가 발생했다.

코드는 아래와 같이 작성했다. 

```swift
    case .review:
      guard
        let appStoreURL = URL(string: "itms-apps://itunes.apple.com/app/id\(myAppID)"), // App Store 앱 직행
        let webURL = URL(string: "https://apps.apple.com/kr/app/id\(myAppID)") // 웹으로 이동(Safari 백업용)
      else { return }

      // 먼저 App Store 시도 → 실패하면 Safari로 연결
      UIApplication.shared.open(appStoreURL, options: [:]) { success in
        if !success {
          UIApplication.shared.open(webURL)
        }
      }
```

<br>

## 왜 실패할까?

찾아보니, iOS 시뮬레이터에는 App Store 앱이 없었다.

즉 App Store 앱이 없으니, 스킴을 처리할 주체가 없어서 `itms-apps://` 방식은 동작하지 않는 것이었다.

<br>

## 실기기로 테스트 진행

실기기로 검증하려고 했지만, iOS 26베타로 업데이트를 해두어서 실기기 테스트가 힘들었다.. 다행히 마침 Slack에 팀원분이 온라인이셔서 실기기 테스트를 부탁드렸다.

## 정리

### **App Store 앱 이동 방식**

---

iOS에서 내 앱(App Store 페이지)을 여는 방법은 크게 두 가지.

1. **App Store 앱 직행 (권장)**

```swift
let appStoreURL = URL(string: "itms-apps://itunes.apple.com/app/id\(appID)")!
UIApplication.shared.open(appStoreURL)
```

- `itms-apps://` → App Store 전용 스킴
- iOS가 바로 **App Store 앱**을 실행해서 해당 앱 상세 페이지로 이동
- UX가 자연스럽고 빠름
- 단, 시뮬레이터에서는 실패하니 실기기를 연결해서 테스트

2. **웹으로 진입**

```swift
let webURL = URL(string: "https://apps.apple.com/kr/app/id\(appID)")!
UIApplication.shared.open(webURL)
```

- 일반 웹 주소
- fallback 용도로 많이 사용

### **리뷰 작성 페이지로 바로 이동**

---

- 리뷰 작성 화면으로 곧장 열고 싶을 때는 **쿼리 파라미터 추가**: `?action=write-review`

```swift
let reviewURL = URL(string: "itms-apps://itunes.apple.com/app/id\(appID)?action=write-review")!
UIApplication.shared.open(reviewURL)
```

### **URL 형태**

---

- **App Store 앱 직행**

```swift
itms-apps://itunes.apple.com/app/id{앱ID}
```

- **웹 주소 (국가 코드 포함 권장)**

```swift
https://apps.apple.com/{country}/app/id{앱ID}
```

- 예: https://apps.apple.com/kr/app/id544007664

<br>

- **리뷰 작성 바로 이동**

```swift
itms-apps://itunes.apple.com/app/id{앱ID}?action=write-review
```

<br>

## `AppID` 확인 방법

App Store 페이지로 이동하려면 반드시 정확한 AppID가 필요하다. AppID는 App Store Connect에서 확인할 수 있다.

[App Store Connect](https://appstoreconnect.apple.com/)

위 주소로 들어가서, “앱” 화면을 들어간다.

<img width="357" height="380" alt="Image" src="https://github.com/user-attachments/assets/b0c5788a-35bb-485e-a33d-5bdb3c3c5a4f" />

앱 목록에서 해당 앱을 선택하여 들어가면 브라우저 주소창에 `apps`뒤에 저런 숫자가 보이는데, 저 숫자가 바로 `AppID`이다.

```swift
.../apps/1234567890/...
```

여기서 1234567890 부분이 바로 AppID이다.

<img width="1580" height="993" alt="Image" src="https://github.com/user-attachments/assets/24e221a8-f55b-4dd0-986c-43b9d22edc3b" />
