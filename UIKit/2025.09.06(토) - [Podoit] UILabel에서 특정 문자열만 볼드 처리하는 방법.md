위 이미지처럼 “**1분 이상**” 부분만 볼드 처리를 디자이너님이 요청하셔서 방법을 찾기 시작했다. 
(위 이미지는 Figma)

처음에는 **UILabel을 2개로 나눠 배치**하려고 했다. 그런데 팀원분이 하나의 UILabel에서도 일부 텍스트만 다르게 할 수 있는걸 봤었던 것 같다고 말해주셔서 관련 방법을 찾아보기 시작했다.

<br>

초기 상태는 아래 이미지와 같이 부분 볼드 처리가 없는 상태였다.

찾아본 결과, `NSMutableAttributedString` 을 활용해서 문자열의 특정 범위에만 스타일을 적용하는 방법을 찾았고, 아래처럼 구현할 수 있었다.

<br>

> **특정 구간 볼드처리 함수** 

```swift
  private func messageLabelTargetBolding(fullText: String, boldTarget: String) -> NSAttributedString {
    // 기본 스타일은 기본 messageLabel의 Typography값으로 전체는 원 상태 유지
    let base = Typography.attributed(
      fullText,
      style: .bodyLg(weight: .regular),
      color: .gray500
    )
    // 수정을 위해 가변 타입인 NSMutableAttributedString으로 변환
    let m = NSMutableAttributedString(attributedString: base)
    
    // 원하는 일부 부분(target)만 볼드처리
    if let range = fullText.range(of: boldTarget) { // boldTarget 구간만 가져옴
      let ns = NSRange(range, in: fullText) // fullText안에서 range의 구간을 nsRange로 변환
      // 원하는 구간만 ".bold"로 덮어쓰기 (다른 부분은 수정되지 않고 base상태)
      let boldFont = Typography.font(for: .bodyLg(weight: .bold))
      m.addAttribute(.font, value: boldFont, range: ns) // 가변으로 만들어준 m에다가 내가 원하는 구간(ns)에 bold(boldFont)를 덮어씀
    }
    return m
  }
```

설명하면, 기본 스타일이 적용된 `attributed string`을 만든 뒤 `NSMutableAttributedString`으로 변환하고, **원하는 구간만 찾아** **해당 범위에 볼드 폰트를 덮어씌워서 반환**하는 방식이다.

<br>

> **NSRange에 대해서 이해하기위해 공부한 내용**
> 

```swift
let ns = NSRange(range, in: fullText) // fullText안에서 range의 구간을 nsRange 타입으로 변환
```

`fullText.range(of: )`로 찾은 Swift String의 Range를 `NSAttributedString`에서 사용할 수 있는 `NSRange`로 변환하는 코드이다.

예를 들어서, `fullText = "1분 이상 집중하세요."`, `boldTarget = "1분 이상"`이라면, 이 예시는 시작 위치가 맨 앞이므로 `location = 0`, `length = 5`가 된다. 

참고로 NSRange는 내부적으로 NSString 기준이기에, 한글/영문은 글자수랑 동일하게 보이지만, 이모지나 조합 문자에서는 `length`가 다를 수 있다고 한다.

<br>

> **적용한 코드**
> 

```swift
messageLabel.attributedText = centered(messageLabelTargetBolding(fullText: message, boldTarget: "1분 이상"))
```

`centered`는 중앙 정렬한거라 특정 부분 볼드랑은 무관하며, `fullText` 에는 전체 텍스트를 전달하고, `boldTarget`에는 그 중에 볼드 처리할 문자열을 넣으면 해당 구간에만 스타일 적용이된다.
